# PulsuProTest
PulsuPro Test Calculator API

## Dependencies
- fastapi
- uvicorn
- requests
- pymongo

## Install
```
git clone https://github.com/archieruin/PulsuProTest.git
cd PulsuProTest
pipenv install
pipenv shell
```

## Run
```
pipenv shell
uvicorn main:app --reload
```

## Tests
`python -m tests`
